# 🚀 Cypress Automation Framework

## 📌 Overview

This project is an end-to-end test automation framework built using **Cypress** and **JavaScript (ES6)**.

It demonstrates real-world automation practices including:

* ✅ Page Object Model (POM)
* ✅ Cross-origin testing using `cy.origin()`
* ✅ Modular and reusable test design
* ✅ CLI-based execution
* ✅ Git-based version control

---

## 🛠️ Tech Stack

* Cypress
* JavaScript (ES6)
* Node.js
* Git & GitHub

## ▶️ How to Run Tests

### Install dependencies

```
npm install
```

### Run all tests (headless)

```
npm run test
```

### Open Cypress Test Runner

```
npm run open
```
